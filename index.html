Programming Assignment

Instructions

Complete this assignment using only the files supplied. You are in charge of the user interface and the user experience. Create buttons (or other elements which accept user interaction) that allow the user to perform the following functions:

filtering
Allow the user to show only those rows in the table which match the filter applied
sorting
Allow the user to sort the table by the column specified (sorting by ascending only is fine).

You are to replace the body of the table – upon page load – with the contents of the XML file which is part of this project. The XML file contains around 142 movies. So the table will contain about 142 movies (un-filtered). You are to use pure ("native") JavaScript. Do not use any external frameworks or libraries such as jQuery, Angular, React, Backbone, Ember, Aurelia, Kendo, etc. You are also encouraged to style the page to "look nicer". Any style changes should be done by modifying the stylesheet included. Here is a suggestion of steps you will need to take to complete this assignment

  • create a Git repository for this web project
  • push your newly initialized Git project to GitHub
  • give the page some basic styling (CSS)
  • make it so that the sample row(s) in the table are replaced with "real"data ‐ the contents of the XML file
  • create functions which provide for sorting and filtering the table of movies
  • test the buttons to make sure they alter the table as desired
  • make periodic commits to your Git repo as you progress
  • push your changes to GitHub
  • let us know when your project is ready for review
